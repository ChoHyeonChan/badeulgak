# 받을각 — 내가 몰라서 못 받는 돈, AI가 찾아드립니다

2026 K-AI 콘텐츠 공모전 **Track B. 솔루션 부문** 출품작입니다.

## 어떤 문제를 해결하나요?

전국에 수백 개의 청년·복지·금융 지원제도가 있지만, 자격이 되는 사람의 상당수가
**존재 자체를 몰라서** 혜택을 놓칩니다. 기존 서비스(보조금24, 복지 앱 등)는
공동인증서 로그인과 관공서 용어가 진입장벽입니다.

**받을각**은 로그인·개인정보 수집 없이, 나이·상황 5문항과 자연어 고민 한 줄만으로
지금 신청할 수 있는 지원제도를 쉬운 말로 찾아줍니다.

## 특징

- 🔒 **개인정보 제로**: 모든 분석이 브라우저 안에서 실행되며 어떤 정보도 서버로 전송·저장되지 않습니다.
- 💬 **자연어 고민 이해**: "월세가 부담돼요"라고 쓰면 그 고민에 맞는 제도부터 보여줍니다.
- 📋 **실존 제도 18개 수록**: 복지로·정부24·고용24 등 공식 출처 기반, 각 결과에 공식 신청처 링크 제공.
- 🧩 **확장 훅 내장**: `netlify/functions/ai-summary.js` — Netlify 배포 + `GEMINI_API_KEY` 등록 시
  Gemini 기반 개인화 상담 요약이 자동 활성화됩니다(없어도 100% 동작).

## 기술

HTML/CSS/JavaScript 단일 정적 웹앱 (프레임워크·빌드 도구 없음).
Claude Code(바이브코딩)로 기획·개발·테스트·배포 전 과정을 진행했습니다.

## 데이터 출처

수록된 지원제도 정보는 복지로, 정부24, 고용24, 한국장학재단, 서민금융진흥원,
주택도시기금, 국세청 홈택스, 큐넷 등 각 제도의 공식 안내 페이지를 참고해 요약했습니다.
세부 자격 요건과 모집 여부는 반드시 공식 신청처에서 최종 확인이 필요합니다.
